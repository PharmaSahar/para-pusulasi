17. Disaster Recovery (Felaket Kurtarma)

VPS tamamen silinse / bozulsa bu adımları uygula. Yaklaşık 30–45 dakika.

Mutlaka yedeklenmesi gereken dosyalar:

DosyaKonumKritiklik
.envMac proje dizini🔴 Tüm API anahtarları
channels/*/youtube_token.pickleMac + VPS🔴 9 kanal OAuth tokeni
channels/channel_registry.jsonMac + VPS🔴 Kanal listesi
~/.ssh/hetzner_vpsMac🔴 SSH private key
PROGRESS.mdMac🟡 Durum kaydı
Kaynak kod (src/)Mac + Git🟢 Git'ten kurtarılabilir

Haftada bir yedek alma (Mac'ten):

cd "/Users/klara/Downloads/adsız klasör"
zip -r ~/Desktop/parapusulasi_backup_$(date +%Y%m%d).zip \
    .env channels/channel_registry.json channels/*/youtube_token.pickle PROGRESS.md
echo "Yedek: ~/Desktop/parapusulasi_backup_$(date +%Y%m%d).zip"

Yeni VPS'e geri yükleme sırası:

1.  Hetzner'da yeni sunucu oluştur (Ubuntu 22.04, CPX32)
2.  SSH key ekle (Hetzner panelinde)
3.  Kodu VPS'e kopyala:
      rsync -avz -e "ssh" "/Users/klara/Downloads/adsız klasör/" root@YENİ_IP:/opt/parapusulasi/ \
        --exclude venv --exclude "output/" --exclude "*.mp4" --exclude "*.mp3"
4.  VPS'te sanal ortam kur:
      cd /opt/parapusulasi && python3 -m venv venv
      source venv/bin/activate && pip install -r requirements.txt
5.  .env ve tokenları kopyala:
      scp .env channels/ root@YENİ_IP:/opt/parapusulasi/
6.  4GB swap ekle:
      fallocate -l 4G /swapfile && chmod 600 /swapfile && mkswap /swapfile && swapon /swapfile
7.  Systemd servisi kur:
      systemctl enable parapusulasi && systemctl start parapusulasi
8.  Watchdog cron ekle:
      (crontab -l; echo '*/5 * * * * /opt/parapusulasi/watchdog.sh') | crontab -
9.  Test: systemctl status parapusulasi
⚠️ SSH Key'i kaybet = tüm kanallara erişim kaybı. ~/.ssh/hetzner_vps dosyasını iCloud / 1Password gibi güvenli bir yere yedekle.
