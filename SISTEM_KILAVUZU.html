  • VPS çalışıyor (systemctl is-active parapusulasi → "active")

12. Yeni Kanal Eklemek

1. YouTube'da kanal aç → URL'den Channel ID al (UC ile başlar)

2. channel_registry.json'a ekle:
   cd "/Users/klara/Downloads/adsız klasör"
   # channels/channel_registry.json dosyasını düzenle

3. OAuth bağla:
   source venv/bin/activate
   python auth.py --channel KANAL_ID

4. VPS'e gönder:
   scp -i ~/.ssh/hetzner_vps channels/KANAL_ID/youtube_token.pickle \
     root@168.119.126.103:/opt/parapusulasi/channels/KANAL_ID/

   scp -i ~/.ssh/hetzner_vps channels/channel_registry.json \
     root@168.119.126.103:/opt/parapusulasi/channels/

5. Servisi yeniden başlat:
   ssh -i ~/.ssh/hetzner_vps root@168.119.126.103 "systemctl restart parapusulasi"

✅ Sistem otomatik tanır ve video render'a başlar.
