# ADR Decision Log

This folder stores architecture decisions that explain why key product-level choices were made.

## How to Use
- Write one ADR per decision.
- Keep Context, Decision, Consequences, and Non-goals explicit.
- Prefer updating ADR status with a follow-up ADR instead of rewriting history.
