# T3.2 - Analytics Collector Contract

## Summary
T3.2 introduces a narrowly scoped Analytics Collector layer that fetches and normalizes YouTube Analytics metrics for Experiment Evaluator consumption.

## Scope
- Fetch per-video analytics metrics from YouTube Analytics API.
- Normalize collector output into evaluator-ready rows.
- Keep collector pure and testable (no pipeline mutation in this step).
- Define explicit error semantics for API and data-shape failures.

## Non-Goals
- No changes to Experiment Evaluator decision algorithm.
- No changes to pipeline runtime behavior.
- No registry write changes.
- No Selection Policy v2/learning logic.
- No scheduler refactor.

## Input Contract
Collector input must include:
- channel_id
- video_ids (list)
- date range or lookback_days

## Output Contract
Collector output row per video must include evaluator fields:
- experiment_id (resolved by caller, not inferred by collector)
- variant_id (resolved by caller mapping)
- impressions
- clicks
- ctr
- watch_time_hours
- average_view_duration_seconds

Optional collector metadata:
- source = youtube_analytics
- fetched_at
- start_date
- end_date
- video_id

## Error Model
- Authentication/API transport failures: surfaced as typed collector errors.
- Empty analytics rows: normalized as null metrics, not hard failure.
- Invalid numeric shape from upstream: rejected with validation error.

## Test Matrix (Minimum)
1. successful_fetch_and_normalize
2. empty_rows_normalized
3. api_error_propagates_typed
4. invalid_metric_shape_rejected
5. deterministic_normalization_for_same_payload

## Commit Boundary
Allowed files for initial T3.2 implementation:
- src/analytics_collector.py
- tests/test_analytics_collector.py

No other files should be modified in the first collector commit.

## Go/No-Go Gate
Before coding collector against live API, probe must show:
1. OAuth success
2. no accessNotConfigured (403)
3. at least one report row returned

If gate is No-Go:
- keep implementation limited to contract docs and test fixtures/mock-first design.
